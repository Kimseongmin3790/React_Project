# 🎮 GClip – 게임 하이라이트 공유 SNS

게임 스크린샷·하이라이트 클립을 올리고 공유하는 **게임 특화 SNS 서비스**입니다.  
유저는 자신이 플레이한 게임의 멋진 순간을 업로드하고,  
태그·게임·탐색 탭을 통해 다양한 클립을 발견할 수 있습니다.

---

## 🧱 Tech Stack

| 영역 | 기술 |
|------|------|
| Frontend | React, React Router, Context API, MUI(Material UI) |
| Backend | Node.js (Express) |
| DB | MySQL |
| 실시간 | socket.io |
| 기타 | JWT 인증, RAWG Games API(게임 목록/정보) |

---

## ⭐ 주요 기능

### 1. 메인 피드

게임별 하이라이트 게시글(이미지/영상) 피드

- **기능**
  - 좋아요 / 북마크 / 공유(링크 복사)
  - 댓글 작성 (기본 댓글, 대댓글/멘션/댓글 좋아요 고도화 예정)
  - 게임 필터 (검색형 드롭다운으로 게임 선택)
  - 정렬/기간 필터 (최신순 등 → 확장 가능)
- **UI**
  - 인스타그램 스타일 카드 레이아웃
  - 유저 아바타/닉네임, 게임 이름, 업로드 날짜, 좋아요/댓글 수 표시
  - 캡션이 길면 ‘더보기’ 버튼으로 상세 모달 이동
- **기타**
  - 피드 카드를 클릭하면 **게시글 상세 모달(PostDetailDialog)** 오픈  
  - 해시태그 `#태그` 자동 추출 및 태그 피드와 연동

---

### 2. 게시글 작성 (CreatePostDialog)

사이드바의 **글 쓰기 버튼** 클릭 시 모달로 게시글 작성

- **기능**
  - 게임 선택 (검색 가능한 드롭다운)
  - 캡션 입력 + 해시태그 인식
  - 이미지 여러 장 업로드
  - 동영상 1개 업로드
  - 업로드 후 메인 피드 자동 새로고침
- **UI**
  - MUI Dialog 기반 모달
  - 이미지/영상 파일 선택 영역 + 프리뷰
  - 유효성 검사 (게임 필수, 미디어 최소 1개 이상 등)

---

### 3. 검색 (Search)

상단 헤더 검색창에서 **통합 검색**

- **검색 대상**
  - 유저 (닉네임/아이디)
  - 클립(게시글)
  - 태그
  - 게임
- **UI**
  - 탭 구성: `통합 / 유저 / 클립 / 태그 / 게임`
  - 유저 카드 클릭 시 → 해당 유저 프로필 페이지 이동
  - 태그 클릭 시 → 태그 피드로 이동 (`/tags/:tagName`)
  - 게임 클릭 시 → 메인 피드에서 해당 게임으로 필터링

---

### 4. 게임 랭킹 (인기 TOP 10 게임)

최근 기간 기준으로 **인기 게임 TOP 10**을 보여주는 페이지

- **지표**
  - 게시글 수
  - 누적 좋아요 수
  - 누적 댓글 수
  - 위 지표를 조합한 점수(score)로 랭킹 계산
- **UI**
  - 순위/게임명/게시글·좋아요·댓글 수 표시
  - LinearProgress를 활용한 점수 막대 그래프
  - 각 게임 카드에 “이 게임 피드 보기” 버튼  
    → 메인 피드로 이동하며 해당 게임 필터 자동 적용
  - (RAWG API에서 가져온 게임 썸네일도 함께 표시)

---

### 5. 탐색 탭 (Explore)

새로운 클립과 태그, 게임을 발견할 수 있는 **탐색 페이지**

- **섹션 구성**
  1. **인기 태그**
     - 최근 N일 기준 사용량이 많은 태그 TOP N
     - `#태그명 · 게시글 수` 형태의 Chip 리스트
     - 클릭 시 → 태그 피드로 이동
  2. **최근 많이 올라오는 게임**
     - 최근 N일 동안 게시글이 많이 올라온 게임 목록
     - 게임 썸네일 + 게시글 수
     - “이 게임 피드 보기” 버튼으로 메인 피드 필터 연동
  3. **랜덤 추천 클립**
     - 랜덤으로 뽑은 하이라이트 그리드
     - 썸네일 클릭 시 → 상세 모달(PostDetailDialog) 오픈

---

### 6. 실시간 채팅

socket.io 기반 **실시간 채팅 기능**

- **기능**
  - 게임별 채팅방
    - 게임 선택 → 해당 게임 전용 채팅방 입장
  - 1:1 DM
    - 유저 검색 후 DM 방 생성/입장
  - 방별 안읽은 메시지 수 표시
  - 최근 메시지 도착 시 상단 알림 박스로 “이 방으로 이동” 버튼 제공
- **UI**
  - 좌측: 게임 선택 / DM 대상 검색 영역
  - 우측: 채팅 메시지 리스트 + 입력창
  - 말풍선 정렬(내 메시지 오른쪽, 상대 왼쪽)
  - 날짜/시간 포맷 (오늘은 시간만, 그 외는 날짜+시간)

---

### 7. 프로필 / 마이페이지

유저의 활동과 통계를 보여주는 **프로필 페이지**

- **기능**
  - 프로필 정보: 아바타, 닉네임, 소개(bio)
  - 팔로우/팔로잉/팔로워 수
  - 해당 유저가 올린 클립 그리드
  - 본인 프로필 편집 (닉네임, 소개, 비밀번호, 아바타 변경)
- **UI**
  - 상단 프로필 카드 + 하단 게시글 그리드(인스타 스타일)
  - 그리드 카드 클릭 시 게시글 상세 모달 오픈

---

### 8. 해시태그 시스템

캡션에서 `#태그` 자동 추출 → 태그 DB와 매핑

- **동작 흐름**
  - 글 작성/수정 시 캡션을 파싱해 태그 목록 추출
  - `tags`, `post_tags` 테이블에 Insert/Sync
  - 태그 페이지(`/tags/:tagName`)에서 해당 태그가 포함된 게시글만 조회
- **활용**
  - 탐색 탭 인기 태그 섹션
  - 추후 태그 기반 추천, 검색 고도화에 사용

---

### 9. 유저 레벨 & 업적 (진행 중)

`user_stats`, `achievements`, `user_achievements` 테이블 기반

- **경험치/레벨**
  - 게시글 작성, 좋아요/댓글을 받았을 때 EXP 누적
  - EXP에 따라 자동 레벨 계산 (`level = floor(exp / 100) + 1`)
  - 상단 헤더 / 마이페이지에서 레벨과 진행도 Progress Bar 표시 예정
- **업적**
  - 예: 첫 글 작성, 좋아요 10개 이상 받기, 특정 게임 클립 많이 올리기 등
  - 조건 만족 시 `user_achievements`에 해금 기록 저장 및 알림

---

### 10. 공통 UI & 다크 모드

- **공통 헤더(MainHeader)**
  - 로고, 검색창, 알림 아이콘, 프로필 아바타
  - 알림 드롭다운 메뉴 (새 포스트, 채팅 알림 등)
  - (진행 중) 헤더 우측에 유저 레벨/EXP Progress Bar 표시
- **사이드바(SideNav)**
  - 메인 / 탐색 / 인기 TOP 10 게임 / 실시간 채팅 / 글 쓰기 / 프로필 / 더보기
  - “더보기” 메뉴 안에 **계정 설정** + **다크 모드 토글**
- **다크 모드**
  - MUI Theme + Context 기반 전역 다크 모드
  - 배경/텍스트/카드/아이콘 색상 테마에 맞게 자동 변경

## 🔐 인증 / 권한

- JWT 기반 로그인/회원 인증
- 미로그인 상태에서는 주요 페이지 접근 제한
- 글 작성/수정/삭제, 좋아요/북마크, 댓글 작성 등은 로그인 필요
- 본인 글만 수정/삭제 가능
- (진행 예정) 신고/차단 기능, 관리자 페이지
